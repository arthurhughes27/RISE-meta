# RISE-meta
Repository for files for the development and implementation of the Rank-Based Identification of High-Dimensional Surrogate Markers Meta-Analysis (RISE-meta) method.

## Preprocessing data structure

The files to preprocess the data can be found in the "./analysis/preprocessing/" folder and should be run in this order. 

preprocessing_clinical.R -> preprocessing_expression.R -> preprocessing_immuneresponse.R -> preprocessing_merging.R
